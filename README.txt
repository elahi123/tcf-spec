RELAX NG Schema for TCF - WebLicht Exchange Linguistic Annotations Format (former D-Spin format).

The format is used for linguistically annotated data exchange within
WebLicht architecture. Contains TextCorpus and Lexicon linguistic
annotations. For more details on the format and supported annotations,
see http://weblicht.sfs.uni-tuebingen.de/weblichtwiki

The distribution contains RELAX NG Schema for TCF, and examples of
linguistic annotations in TCF TextCorpus and Lexicon. For the
information on reading/writing TCF data in Java see information on
wlfxb library at http://weblicht.sfs.uni-tuebingen.de/weblichtwiki
